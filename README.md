# 开局托儿所风格页面识别与交互自动化（Academic Demo）

> 本项目用于研究 **屏幕视觉感知（Computer Vision）** 与 **人机交互自动化（UI Automation）** 的工程实现方法，
> 通过对 *自研/授权* 的托儿所风格 UI 界面进行识别、决策与交互执行，验证“识别 → 规划 → 执行”的闭环可行性。
